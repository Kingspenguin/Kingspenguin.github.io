# Jianhan Ma Homepage

Source for [kingspenguin.github.io](https://kingspenguin.github.io/), Jianhan (Jaime) Ma's academic homepage.

The site is a static GitHub Pages page with no build step. It includes profile information, publications, selected projects, education, teaching, and social/SEO metadata.

## Local Preview

Open `index.html` directly in a browser, or serve the directory with any static file server.

## Assets

Only the assets used by the current homepage, plus source media for generated previews, are kept in the repository. Legacy template files have been removed from the working tree.
